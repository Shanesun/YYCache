YYCache.m

Overview

YYCache is a thread safe key-value cache.

It use YYMemoryCache to store objects in a small and fast memory cache, and use YYDiskCache to persisting objects to a large and slow disk cache. See YYMemoryCache and YYDiskCache for more information.
